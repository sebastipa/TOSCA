Time Integration Schemes

Four time-stepping schemes are available for both the momentum and temperature equations. The momentum scheme is selected via the keyword -dUdtScheme and the temperature scheme via -dTdtScheme in control.dat.

Momentum Equation

Let \(\mathcal{H}^q_U = H^q_U + L^q\) denote the full right-hand side of the momentum equation (advection, viscous diffusion, and all source terms, excluding the pressure gradient). The pressure gradient \(G^q(p^n)\) is always treated explicitly and removed by the projection step (see Numerical Method).

Forward Euler

First-order explicit scheme:

\[V^{q,n+1}_* = V^{q,n} + \Delta t\,\left[\mathcal{H}^q_U(V^n, u^n) - G^q(p^n)\right].\]

Conditionally stable; requires \(\mathrm{CFL} < 1\). Recommended only for coarse (fast) exploratory runs.

Runge–Kutta 4

Classical four-stage explicit scheme:

\[V^{q,n+1}_* = V^{q,n} + \Delta t\sum_{s=1}^{4} b_s K_s, \qquad K_s = \mathcal{H}^q_U\bigl(V^n + a_s\,\Delta t\,K_{s-1},\,u^n + a_s\,\Delta t\,K_{s-1}\bigr),\]

with Butcher coefficients \(\mathbf{b} = (\tfrac{1}{6},\tfrac{1}{3},\tfrac{1}{3},\tfrac{1}{6})^\top\) and \(\mathbf{a} = (0,\tfrac{1}{2},\tfrac{1}{2},1)^\top\). Fourth-order accurate in time; conditionally stable. Requires four full right-hand-side evaluations per step.

Crank–Nicolson

Fully implicit second-order scheme solved with PETSc’s Newton trust-region SNES solver. The update reads:

\[\frac{V^{q,n+1}_* - V^{q,n}}{\Delta t} = \frac{1}{2}\,\mathcal{H}^q_U(V^{n+1}_*, u^{n+1}_*) + \frac{1}{2}\,\mathcal{H}^q_U(V^n, u^n) - G^q(p^n).\]

The nonlinear system is solved using Newton–Krylov iterations with a matrix-free Jacobian (MATMFFD) and a BiCGStab or GMRES (more robust slower) inner solver. To reduce the number of Newton iterations, a source buffer \(\mathbf{b}_U\) precomputes all terms that are independent of the current iterate (pressure gradient, \(\tfrac{1}{2}\mathcal{H}^q_U(V^n,u^n)\), source terms), and an explicit Forward Euler prediction provides the initial guess. The scheme is unconditionally stable and second-order accurate in time. Recommended for production runs where the cost of nonlinear solves is offset by the ability to use large time steps (CFL \(\to 1\)).

IMEX-CNAB (Implicit–Explicit Crank–Nicolson Adams–Bashforth)

A split scheme where convection is treated explicitly with Adams–Bashforth 2 (AB2) and viscous diffusion is treated implicitly with Crank–Nicolson. The update reads:

\[\frac{V^{q,n+1}_* - V^{q,n}}{\Delta t} = \mathbf{b}^q_U + \tfrac{1}{2}\,\mathcal{L}^q(V^{n+1}_*),\]

where the explicit buffer \(\mathbf{b}^q_U\) is assembled once per step from fields at time levels \(n\) and \(n-1\):

\[\mathbf{b}^q_U = -G^q(p^n) + L^{q,\text{buoy}} + \tfrac{3}{2}\,\mathcal{N}^q_U(V^n, u^n) - \tfrac{1}{2}\,\mathcal{N}^q_U(V^{n-1}, u^{n-1}) + \tfrac{1}{2}\,\mathcal{L}^q(V^n) + \mathcal{S}^q,\]

with \(\mathcal{N}^q_U\) the convective operator, \(\mathcal{L}^q\) the viscous-diffusion operator, and \(\mathcal{S}^q\) the remaining source terms. The first step uses Forward Euler for the AB2 convection stencil. The resulting linear system

\[A\,V^{q,n+1}_* = V^{q,n} + \Delta t\,\mathbf{b}^q_U, \qquad A\,v \equiv v - \tfrac{\Delta t}{2}\,\mathcal{L}^q(v),\]

is solved directly with a standalone PETSc KSP (GMRES or BiCGStab, with \(L=2\)) solver. The scheme is second-order accurate in time and overall faster than Crank-Nicholson in terms of time advancement of the simulation. However, because convection is treated explicitly, the scheme is only conditionally stable and often requires stabilization through the -central4 + -hyperVisc combination (see Advection Schemes) or the addition of biharmonic hyperviscosity (see IMEX-CNAB Biharmonic Hyperviscosity below) to suppress Nyquist-mode noise at high CFL numbers. The recommended CFL limit is \(\mathrm{CFL} \lesssim 0.5\), if no explicit filtering is used, or \(\mathrm{CFL} \lesssim 0.6\) when biharmonic hyperviscosity is active (either as a standalone correction or as a mild filter on top of the -central4 scheme). Running this scheme with CFL close to or above 1.0 will lead to numerical instability.

IMEX-CNAB Biharmonic Hyperviscosity

For the IMEX-CNAB scheme, an optional biharmonic (fourth-order) hyperviscosity correction is available to suppress odd–even checkerboard noise and instability at the Nyquist wavenumber. The per-step flux correction is

\[\Delta V^q_{\mathrm{hv}} = -\varepsilon_4 \bigl(\delta^4_\xi + \delta^4_\eta + \delta^4_\zeta\bigr)\,V^{q,n},\]

where the one-dimensional fourth-order difference operator is

\[\delta^4_d\,v_n = v_{n+2} - 4v_{n+1} + 6v_n - 4v_{n-1} + v_{n-2},\]

with Fourier symbol \(\widehat{\delta^4}(\kappa) = 4(\cos\kappa h - 1)^2\). The symbol vanishes at \(\kappa h = 0\) (no effect on resolved scales) and reaches 16 at the Nyquist wavenumber \(\kappa h = \pi\). For stable, non-amplifying operation the coefficient must satisfy

\[\varepsilon_4 \leq \frac{1}{48} \approx 0.021.\]

The coefficient is directional and it is set by the keywords -imexHyperVisc4U_i, -imexHyperVisc4U_j and -imexHyperVisc4U_k in control.dat. When hyperviscosity is active, the effective CFL limit for IMEX-CNAB increases from 0.5 to approximately 0.6-0.7. Notably, hyperviscosity addition can be directional, allowing for selective damping of the Nyquist mode in the horizontal or vertical directios.

Temperature Equation

Let \(\mathcal{F}(T) = \mathcal{N}(T) + \mathcal{D}(T)\) be the full right-hand side of the temperature equation (convection \(\mathcal{N}\) plus diffusion \(\mathcal{D}\)), including Rayleigh-damping and source terms.

Runge–Kutta 4

Four-stage explicit scheme, identical in structure to the momentum counterpart:

\[T^{n+1} = T^n + \Delta t\sum_{s=1}^{4} b_s K_s, \qquad K_s = \mathcal{F}\!\bigl(T^n + a_s\,\Delta t\,K_{s-1}\bigr),\]

with the same Butcher coefficients as above. Conditionally stable; requires \(\mathrm{CFL} < 1\).

Backward Euler

Fully implicit first-order scheme. The nonlinear residual

\[\mathcal{G}(T^{n+1}) = -\frac{T^{n+1} - T^n}{\Delta t} + \mathcal{F}(T^{n+1}) = 0\]

is solved using PETSc’s SNES with the same Newton–Krylov configuration used for the momentum equation. Unconditionally stable; first-order accurate in time.

Crank–Nicolson

Second-order implicit scheme that averages \(\mathcal{F}\) at the old and new time levels:

\[\frac{T^{n+1} - T^n}{\Delta t} = \tfrac{1}{2}\,\mathcal{F}(T^{n+1}) + \tfrac{1}{2}\,\mathcal{F}(T^n).\]

The explicit contribution \(\tfrac{1}{2}\mathcal{F}(T^n)\) is precomputed and stored in the buffer Rhs_o before the SNES nonlinear solve, which uses the same Newton–Krylov configuration as Backward Euler. Unconditionally stable and second-order accurate in time.

IMEX-BEAB (Implicit–Explicit Backward Euler Adams–Bashforth 2)

Implicit–explicit scheme: convection treated explicitly with Adams–Bashforth 2 and diffusion treated implicitly with Backward Euler. The scheme avoids the need for \(\mathbf{u}^{n+1}\) and yields a linear system in \(T^{n+1}\):

\[\underbrace{\bigl(I - \Delta t\,\mathcal{D}\bigr)}_{A}\,T^{n+1} = T^n + \Delta t\,b_T,\]

where the explicit buffer is

\[b_T = c_1\,\mathcal{N}(T^n) + c_2\,\mathcal{N}(T^{n-1}) + \mathcal{D}_\text{damp}(T^n) + \frac{s_T}{\Delta t},\]

with \(c_1 = 3/2\), \(c_2 = -1/2\) (AB2; on the first step: \(c_1 = 1\), \(c_2 = 0\)), \(\mathcal{D}\) the diffusion operator, \(\mathcal{D}_\text{damp}\) the Rayleigh damping operator, and \(s_T\) a prescribed source term. The operator \(A\) is symmetric positive definite; the system is solved with the same standalone KSP (GMRES or BiCGStab) used for the IMEX momentum equation.
