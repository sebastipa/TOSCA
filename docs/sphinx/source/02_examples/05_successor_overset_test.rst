Wind Farm Successor Example - Overset Mesh
------------------------------------------

The *tests/SuccessorOversetTest* example builds on the *tests/SuccessorPeriodizedTest* example case by adding an overset 
mesh to increase the mesh resolution around the wind turbines. Details regarding the overset mesh algorithm implemented 
in TOSCA can be found in Section :ref:`overset-section` of the documentation.
